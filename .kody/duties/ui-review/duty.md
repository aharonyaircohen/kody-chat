# UI Review

## Job

Review a UI change using the running preview and report visual or interaction issues.

## Executable

Run the `ui-review` executable. Its UI review skills own the detailed method.

## Output

A UI review verdict and actionable findings.

## Allowed Commands

- Run the `ui-review` executable.

## Restrictions

- Do not edit code from this duty.
- Base findings on observed UI behavior.
- Keep recommendations concrete and tied to the preview.
