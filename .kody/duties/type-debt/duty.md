# Type Debt Tracker

## Job

Track TypeScript escape hatches and escalate meaningful growth week over week.

## Executable

Run the `type-debt` executable. Its skill owns the detailed method and runtime state handling.

## Output

A weekly type-debt tracking issue or nudge.

## Allowed Commands

- Run the `type-debt` executable.

## Restrictions

- Do not edit code directly.
- Do not count by ad-hoc local grep from the duty.
- One issue or comment per tick.
- Use the prior count history from runtime state.
