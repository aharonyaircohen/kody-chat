# Memory Compaction

Memory compaction reduces repeated or noisy memory into smaller, safer summaries.

The live proposal report is `.kody/reports/memory-compaction.md` on the `kody-state` branch.

Use it before changing `.kody/memory/`.

Rules:

- Do not compact all memory into one file.
- Keep memory split by purpose.
- Preserve source links or hashes.
- Write a proposal first.
- Apply changes only after review.
